## 通用规格

### 图片格式
- **格式**: PNG (透明背景优先), JPG (背景图)
- **色彩模式**: RGBA (透明), RGB (不透明)
- **推荐分辨率**: 72dpi - 150dpi (屏幕显示)

### 音频格式
- **格式**: OGG Vorbis
- **采样率**: 44.1kHz
- **声道**: 单声道 (SFX), 立体声 (BGM)

### 命名规范
- 使用中文拼音或英文
- 单词间用下划线连接: `xiao_hu_li.png`
- 避免特殊字符和空格
